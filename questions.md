## Context Setting
Establish what was already done before this A/B test. What are some questions you can ask to get more information on what has already been measured, analyzed, or attempted? List them below...

- Have you run any analysis on viewing times before this redesign, or is this the first time you're tracking this?
- Was there any feedback from employees about the original slides before you decided to redesign them?
- How long has the original slide format been in use, and has anything else about the workshop changed recently besides the word count?
- Were both groups watching the same workshop content, or did the topics differ between control and treatment?
- Has viewing time been used as a success metric before, or is this a new way of measuring engagement?


## Goal Alignment
Align with stakeholders on what success means before interpreting any numbers.

- What does a successful outcome look like for you — is it just higher viewing time, or does engagement need to translate into something like quiz scores or job performance?
- Is there a specific threshold you had in mind, like "if the treatment group watches 5 more minutes on average, that's a win"?
- Who is this analysis being presented to, and what decision are they trying to make with these results?
- If the treatment group shows higher viewing time but more dropoff at the end, would that still be considered a success?


## Define Over-performance
Consider what it would mean if the treatment dramatically exceeded expectations.

- If the treatment group's viewing time is significantly higher than expected, would that raise any concerns about the data quality or how the experiment was set up?
- Is there a point where results seem too good — for example, if employees in the treatment group watched twice as long, would that suggest something other than slide design is driving the difference?
- If overperformance is confirmed, is the company prepared to roll out the redesigned slides to all future workshops right away?


## Data Sourcing
Reflect on what other data sources you can use to answer your stakeholders exploratory goals.

- Do you have quiz or assessment scores tied to these same employees that we could compare against viewing time?
- Is there any attendance or completion data that tracks whether employees finished the full workshop or dropped off early?
- Do you have viewing data from previous workshops using the old slides that we could use as a baseline comparison?
- Are there any employee satisfaction surveys from after the workshop that could help explain the difference in engagement?
